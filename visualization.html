Real-Time AQI Visualization

Interactive data visualizations from the Real-Time Air Quality Index (AQI) project

Project Overview

This page showcases the data visualizations created from my Real-Time Air Quality Index (AQI) monitoring project. The visualizations demonstrate patterns, trends, and insights derived from air quality data collected from various monitoring stations across multiple cities.

The project uses Python with libraries such as Matplotlib, Seaborn, and Plotly for creating interactive and informative visualizations that help understand air quality patterns, identify pollution hotspots, and analyze temporal trends.

GitHub Repository: https://github.com/suhaspuli24123/Real_Time_AQI

Key Visualizations

AQI Trends Over Time

This time-series visualization shows the daily average Air Quality Index (AQI) trends over a 6-month period. The plot reveals seasonal patterns and critical periods of high pollution levels, demonstrating how air quality fluctuates over time.

Notable observations include higher pollution levels during winter months (likely due to increased heating and atmospheric conditions) and specific pollution events that correlate with known environmental incidents.

Python Code


# Time series plot with Matplotlib
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# Load and prepare data
aqi_data = pd.read_csv('aqi_daily_data.csv')
aqi_data['date'] = pd.to_datetime(aqi_data['date'])

# Create figure and plot time series
plt.figure(figsize=(12, 6))
plt.plot(aqi_data['date'], aqi_data['aqi_value'], marker='o', linestyle='-', color='#1e88e5')
plt.axhline(y=100, color='r', linestyle='--', alpha=0.7, label='Unhealthy Threshold')

# Add labels and formatting
plt.title('Daily Average AQI Over Time', fontsize=16)
plt.xlabel('Date', fontsize=12)
plt.ylabel('AQI Value', fontsize=12)
plt.grid(True, alpha=0.3)
plt.legend()
plt.tight_layout()

# Save the figure
plt.savefig('aqi_time_series.png', dpi=300, bbox_inches='tight')
plt.show()

Pollutant Concentration by Location

This bar chart compares the average concentration of different pollutants (PM2.5, PM10, NO2, SO2, CO, O3) across multiple monitoring stations. The visualization helps identify which pollutants are more prevalent in specific areas.

The data indicates that urban areas show higher levels of NO2 and PM2.5, consistent with traffic-related pollution, while industrial zones show elevated levels of SO2 and PM10.

Python Code


# Bar chart with Seaborn
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# Load and prepare data
pollutant_data = pd.read_csv('pollutant_by_location.csv')

# Create the bar chart
plt.figure(figsize=(14, 8))
chart = sns.barplot(
    x='location', 
    y='concentration', 
    hue='pollutant',
    palette='viridis',
    data=pollutant_data
)

# Add labels and formatting
chart.set_title('Average Pollutant Concentration by Location', fontsize=16)
chart.set_xlabel('Monitoring Station Location', fontsize=12)
chart.set_ylabel('Concentration (μg/m³)', fontsize=12)
plt.xticks(rotation=45, ha='right')
plt.legend(title='Pollutant')
plt.tight_layout()

# Save the figure
plt.savefig('pollutant_bar_chart.png', dpi=300, bbox_inches='tight')
plt.show()

Correlation Heatmap of Pollutants

This heatmap visualizes the correlation between different pollutants. Understanding these relationships helps identify common sources and can inform more targeted pollution control strategies.

The heatmap reveals strong positive correlations between PM2.5 and PM10, suggesting common sources or formation mechanisms. Additionally, NO2 and CO show significant correlation, indicating traffic as a likely common source.

Python Code


# Correlation heatmap with Seaborn
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

# Load and prepare data
pollutant_data = pd.read_csv('pollutants_daily.csv')

# Calculate correlation matrix
correlation = pollutant_data[['PM2.5', 'PM10', 'NO2', 'SO2', 'CO', 'O3']].corr()

# Create the heatmap
plt.figure(figsize=(10, 8))
mask = np.triu(correlation)
heatmap = sns.heatmap(
    correlation,
    annot=True,
    cmap='coolwarm',
    vmin=-1,
    vmax=1,
    mask=mask,
    square=True,
    linewidths=0.5,
    cbar_kws={'shrink': 0.8}
)

# Add labels and formatting
heatmap.set_title('Correlation Between Air Pollutants', fontsize=16, pad=20)
plt.tight_layout()

# Save the figure
plt.savefig('pollutant_correlation_heatmap.png', dpi=300, bbox_inches='tight')
plt.show()

Geographic Distribution of AQI

This interactive map visualization displays the geographic distribution of Air Quality Index (AQI) values across different monitoring stations. The map helps identify pollution hotspots and spatial patterns in air quality.

The visualization clearly shows higher pollution levels in densely populated urban areas and around industrial zones, with cleaner air in suburban and peripheral regions.

Python Code


# Interactive map with Plotly
import plotly.express as px
import pandas as pd

# Load and prepare data
geo_data = pd.read_csv('aqi_geo_data.csv')

# Create the interactive map
fig = px.scatter_mapbox(
    geo_data,
    lat='latitude',
    lon='longitude',
    color='aqi_value',
    size='aqi_value',
    color_continuous_scale=px.colors.sequential.Plasma,
    size_max=15,
    zoom=10,
    mapbox_style='carto-positron',
    hover_name='location',
    hover_data={
        'aqi_value': True,
        'main_pollutant': True,
        'latitude': False,
        'longitude': False
    },
    title='Geographic Distribution of Air Quality Index'
)

# Update layout
fig.update_layout(
    margin={'r': 0, 't': 50, 'l': 0, 'b': 0},
    coloraxis_colorbar={
        'title': 'AQI Value',
        'tickvals': [50, 100, 150, 200, 300],
        'ticktext': ['Good', 'Moderate', 'Unhealthy for Sensitive Groups', 'Unhealthy', 'Very Unhealthy']
    }
)

# Save as HTML file
fig.write_html('aqi_map.html')

# Show the plot
fig.show()
